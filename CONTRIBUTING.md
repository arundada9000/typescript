# Contributing

This is a personal TypeScript learning repository, but contributions and suggestions are welcome!

## How to Contribute

1. Fork the repository
2. Create a new branch (`git checkout -b feat/your-idea`)
3. Make your changes
4. Ensure TypeScript compiles without errors (`npx tsc --noEmit`)
5. Commit with a descriptive message
6. Push and open a Pull Request

## Guidelines

- Keep examples clear and beginner-friendly
- Follow the existing chai/tea theme if adding examples
- Add comments explaining the TypeScript concept being demonstrated
- Update the README if adding new topics

## Code Style

- Use TypeScript with strict mode (already configured)
- No unused variables or parameters
- Use `const` over `let` where possible
- Prefer explicit return types on functions
