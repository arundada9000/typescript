# Security Policy

## Reporting a Vulnerability

This is a learning repository with no production deployment. If you find a security issue:

1. **Do not open a public issue** - email [arunneupane0000@gmail.com](mailto:arunneupane0000@gmail.com)
2. Include a description of the issue and steps to reproduce
3. Allow reasonable time for a response before any disclosure

## Supported Versions

The latest commit on `main` is the only supported version.
